# @spaguettiscope/plugin-drizzle

Drizzle ORM scan plugin for SpaguettiScope. Classifies Drizzle schema, migration, and query files
into topology dimensions.

```json
{ "plugins": ["@spaguettiscope/plugin-drizzle"] }
```

See the [main repository](https://github.com/jasonsantos/spaguettiscope) for full documentation.
